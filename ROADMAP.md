## Vision

Visual build tool for automating client/feature site creation with a dedicated client/server architecture.

## Current State

- **Status**: Repo scaffold present (client/server + docker-compose)
- **Roadmap**: Not documented

## Phase Overview

- **Phase 1 (Core)**: Foundation and MVP scope
- **Phase 2 (Expansion)**: Feature depth and integrations
- **Phase 3 (Scale/Polish)**: Reliability, automation, and UX polish

- Reusable blocks
- Template presets

- Multi-page sites
- Navigation builder

- Global tokens
- Per-component overrides
