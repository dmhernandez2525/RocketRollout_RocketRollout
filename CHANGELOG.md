## [0.2.0] 
### Added

- StylingSettings: Default component to hold all of the default stylings for all components ex: Layout, Spacing, Size, Position, Typography,     Background, Borders, Effects, ClassSelector
- Layout: Component to hold and modify css layout options
- Spacing: Component to hold and modify css spacing options
- Size: Component to hold and modify css size options
- Position: Component to hold and modify css position options
- Typography: Component to hold and modify css typography options
- Background: Component to hold and modify css background options
- Borders: Component to hold and modify css borders options
- Effects: Component to hold and modify css effects options
- ClassSelector: Component to hold and modify the class selector
